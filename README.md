# Contact Form with Nodemailer

## Local Development

Clone the repo.

Install dependencies:
```
yarn
```
OR
```
npm install
```

Add a .env file with the following info:
```
EMAIL_USERNAME="youremail@gmail.com"
EMAIL_PASSWORD="YoUrPa$$w0Rd"
```
Run it.
```
npm start
```
Try it.<br>
Go to `http://localhost:3000/` in your favorite browser.
